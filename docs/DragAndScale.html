Class: DragAndScale

DragAndScale(element, skip_events)

Class responsible for handling scale and offset transformations for an HTML element, enabling zooming and dragging functionalities.

Constructor

new DragAndScale(element, skip_events)

Creates an instance of DragAndScale.
Parameters:
Name Type Description
element HTMLElement The HTML element to apply scale and offset transformations.
skip_events boolean Flag indicating whether to skip binding mouse and wheel events. Rendering: toCanvasContext() is HTMLCanvas, and onredraw is probably also. The rest is all HTML+CSS+JS
